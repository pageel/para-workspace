# v1.7.2 — KI Index Schema Upgrade, Workflow Simplification & Knowledge Graph Seed

> **Release Date:** 2026-04-02
> **Type:** PATCH (heuristic-level changes only)
> **Kernel:** 1.5.3 (unchanged)

## Summary

This release completes three improvements to the Knowledge System:

1. **Index Schema Upgrade** — Save Index schema expanded from 7 to 12 columns, giving users visibility into Owner, Scope, Domain, Purpose, and PARA Version for each KI.

2. **Workflow Simplification** — Removed redundant `index.md` file reads from 6 workflows. The Antigravity platform auto-injects KI summaries at session start, making file-based loading unnecessary.

3. **Knowledge Graph Seed** — Populated `code_refs` with 16 file-level entries and `concepts` with 10 enriched entries in the system KI template, establishing the foundational data layer for future Knowledge Graph capabilities.
